This is an academic records Portal strictly meant for the lecturer use only
It is built by xml and php plus some htmland css 
It is a dynamic webpage that allowas lecturers to log in and view their students' academic performance across course units
It is a simple one but functional as it demonstrates how php can read structured XML data and render it dynamically as a web page without the need of a database
when a lecturer logs in with their credentials the system loads only the course units assigned to them, displaying each student's exam marks , assignment marks, submission status, computed totals, grades and pass or fail status
